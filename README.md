
# 시각 장애인을 위한 음성주문 기반 키오스크 어플리케이션  '여기다'  

챗봇, TTS, STT를 결합하여 시각 장애인이 음성으로 주문을 할 수 있도록 함.

APP 전반에 시각 장애인을 위한 장애인 접근성 지침을 준수하여 VoiceOver를 이용해 모든 기능을 사용할 수 있음.
